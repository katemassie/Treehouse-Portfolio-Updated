Hello, I’m Kate.

My home is Richmond, Virginia.

Background

I'm a student of web design seeking to further my skills and knowledge daily. I earned a PhD in English literature from the University of North Carolina in Chapel Hill, a process that taught me to be persistent, curious, and detail-oriented. I bring those qualities to web work, which requires tenacious regular interest.

Since I defended my dissertation, I’ve worked as a grantwriter and as an assistant in a flower shop. I fell in love with coding in 2015 and have begun to make learning web design part of my daily life.

Goals

I want to soak up as much information as I can as an apprentice, becoming proficient in:

  • HTML
  • CSS
  • JavaScript
  • PHP
  • Ruby

I like to create, whether that's using my camera, using yarn and a crochet needle, or using HTML and CSS. My ideal job is with a web design firm, crafting beautiful and useful sites for clients.
